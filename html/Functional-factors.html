Functional Specifications

Electronics for the active antenna must be designed with the following functional specifications

  • The design must be compatible with an active whip antenna of 50cm length and 5mm diamter
  • The active antenna must receive power and communicate via the same 50 Ohm coax cable with SMA connectors which is connected to the interface
  • The supply voltage must be less than 10V DC.
  • The input of the active antenna should be protected against ESD discharge
  • The electronics should withstand temperatures between -25 and 70 degrees Celcius
  • The active antenna must contain an amplifier
